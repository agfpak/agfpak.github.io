Альтернативная установка psycopg2 в virtualenv для Postgres.

Устанавливаем psycopg не в окружении через

$ sudo pip install psycopg2

Смотрим куда это встало

$ sudo python -c "import psycopg2;print psycopg2.__file__"
$ /usr/lib/python2.7/dist-packages/psycopg2/__init__.pyc

И копируем к себе в lib’ы в virtualenv

$ sudo cp -r /usr/lib/python2.7/dist-packages/psycopg2  mypythonenv/lib/python2.7/site-packages/

Копипастим яйцо

$ sudo cp /usr/lib/python2.7/dist-packages/psycopg2-2.*.*.egg-info  mypythonenv/lib/python2.7/site-packages/

Еще надо будет скопировать mx

$ sudo cp -r /usr/lib/python2.7/dist-packages/mx  mypythonenv/lib/python2.7/site-packages/
