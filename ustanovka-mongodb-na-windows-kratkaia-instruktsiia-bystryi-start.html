Установка MongoDB на Windows — Краткая инструкция, быстрый старт

  • By agfpak
  • Sun 07 June 2015

  1. Скачиваем и с сайта диструбтив https://www.mongodb.org/downloads и устанавливаем его.

  2. Добавляем в переменные окружения Windows путь к исполняемым файлам установленной базы (C:\Program Files\MongoDB\Server\3.0\bin)

  3. Создаем две папки командой в терминале «md \data» и «md \data\db»

  4. Базу данных запускаем из терминала командой mongod
